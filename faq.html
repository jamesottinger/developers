Frequently Asked Questions

How long do Linodes spun up for alpha testing stay up before deleting?

The alpha environment is wiped every Sunday night.

Will I be charged for Linodes used with the alpha API?

No. You are not charged for anything while using the alpha version of the API.

Is it possible to use the alpha API with my current Linode account?

No, it requires you to request access via alpha.linode.com. See our Engineering Blog for more information.
